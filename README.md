# Code Render Portfolio

Welcome to the official portfolio website for **Code Render** – a one-man software development agency.

This project is a high-performance, modern single-page application built to showcase past projects, technical expertise, and provide a seamless way for potential clients to get in touch.

## 🚀 Tech Stack

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS / Tailwind CSS principles
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router](https://reactrouter.com/)
- **Forms:** [FormSubmit.co](https://formsubmit.co/) (AJAX Integration)

## ✨ Features

- **Modern & Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Infinite Ticker Animation:** Smooth, CSS-driven infinite marquee for the tech stack showcase.
- **Dynamic Routing:** Scalable route setup using React Router for future pages.
- **Secure Contact Form:** Integrated with FormSubmit with HTML5 validations for Name, Email, and Phone Number fields.
- **One-Man Agency Highlight:** Authentic messaging tailored for solo-developer branding.

## 🛠️ Quick Start

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sribalan98/coderender-portfolio.git
   cd coderender-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in Browser:**
   Open [http://localhost:3000](http://localhost:3000) (or whichever port Vite assigns) to view it in the browser.
